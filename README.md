# Weather Checking Web App
Live Demo:
https://wheather-app-naveen.vercel.app/

Description:
A simple and responsive Weather Checking Web Application that allows users to search for any city and view real-time weather information including climate image, temperature, wind speed, and humidity.
The application is built using HTML, CSS, and JavaScript and fetches live weather data from the OpenWeather API.

## Features

1. Search weather by city name  
2. Displays temperature in Celsius  
3. Shows wind speed  
4. Displays humidity percentage  
5. Dynamic climate images based on weather conditions  
6. Error message for invalid city names  
7. Responsive design for all screen sizes  

## Technologies Used

HTML is used for structuring the web application  
CSS is used for styling and responsiveness  
JavaScript is used for API integration and dynamic content updates  
OpenWeather API is used to fetch real-time weather data  

## API Information

OpenWeather API  
Website https://openweathermap.org/api  

The API provides real-time temperature, humidity, wind speed, and climate conditions.
